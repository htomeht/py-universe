The Greedy Gargoyle

by Joe Strout 5/11/96
The Greedy Gargoyle is a 5-Minute Text Puzzle. It consists of a single source file (gredgar.py), which requires the standard PUB library to run. (No fair peeking at the source before you've solved the puzzle!)
